Customer Churn Analysis

Customer churn is a critical challenge for businesses. This project predicts which customers are likely to churn based on factors like service usage, contract type, and customer service interactions.

Data Collection

The dataset consists of customer demographics, subscription details, usage patterns, customer service interactions, and churn status.

Data Preprocessing

Missing data was imputed and outliers were handled. Features like tenure and customer support interactions were created. Categorical variables were one-hot encoded.

Exploratory Data Analysis (EDA)

Customers on month-to-month contracts had higher churn rates, while annual contracts had the lowest churn. High customer service interactions also increased churn probability.

Model Building

Logistic Regression was used as a baseline. Random Forest and XGBoost were applied to handle non-linear relationships. Hyperparameter tuning was performed using GridSearchCV.

Model Evaluation

Accuracy: 87%, AUC-ROC: 0.92. Precision: 0.85, Recall: 0.81. The model balanced between false positives and false negatives effectively.

Key Insights

Contract type and customer service interactions are strong predictors of churn. Businesses can reduce churn by offering incentives for annual contracts and improving customer support.
